# portfolio

Risva's personal portfolio — a small Next.js site built with the App Router, Tailwind CSS and shadcn/base-ui components.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. Resume content lives in `lib/data.ts`.

## Deploy

The easiest way to deploy this app is via the [Vercel Platform](https://vercel.com/new).
